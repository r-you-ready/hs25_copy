## 📖 About

This repository is the course archive for the **"R u Ready"** methods seminar, part of the Psychology of Digitalization curriculum at the University of Bern. It contains all presentation slides and R code used throughout the course.

The R u Ready seminar was developed to equip psychology students with practical data analysis skills using the open-source programming language R — no prior coding experience required.

---

## 📂 Repository Contents

| Folder / File | Description |
|---|---|
| `presentations/` | Slides for each session of the course |
| `code/` | R scripts and exercises used in the seminar |
| `index.html` | Source for the course website |

## 🛠️ Prerequisites

To run the R code in this repository, you will need:

- **R** (≥ 4.0 recommended)
- **RStudio** (recommended IDE)

> Relevant R packages are specified within each individual script.
